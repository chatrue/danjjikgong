# 📘 단찍공 (DanJjikGong)

단찍공은 사진으로 촬영한 단어장을 OCR로 인식하여
영어 단어와 뜻을 빠르게 정리하고 학습할 수 있도록 만든 모바일 중심 학습 앱입니다.

복잡한 서버 없이 **브라우저만으로 동작**하도록 설계되어 누구나 가볍게 사용할 수 있습니다.

---

## ✨ 주요 기능

* 📸 **사진 OCR 인식**

  * 촬영한 단어장 이미지에서 영어/한국어 자동 추출
* 🧠 **단어 학습 모드**

  * 객관식 퀴즈
  * 주관식 입력
* 🔊 **TTS 발음**

  * 단어 음성 재생
* 📱 **모바일 최적화 UI**

  * 태블릿 및 스마트폰 사용에 맞춘 레이아웃
* 💾 **로컬 저장**

  * 서버 없이 브라우저에 데이터 저장

---

## 🧩 기술 스택

* React + Vite
* Tesseract.js (OCR)
* Web Speech API (TTS)
* PWA 기반 구조

---

## 🚀 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 아래 주소 접속:

```
http://localhost:5173
```

## 📱 사용 방법

1. 단어장 사진 촬영 또는 업로드

## ⚠️ 참고 사항

* OCR 정확도는 이미지 품질에 따라 달라질 수 있습니다.
* 서버를 사용하지 않기 때문에 데이터는 기기 브라우저에 저장됩니다.
